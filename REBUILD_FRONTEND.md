1. Open File Explorer
2. Navigate to: `ai-learning-assistant`
3. **Delete the `frontend` folder completely**
4. Empty Recycle Bin

### **STEP 3: Create New Vite + React Frontend**

Open Command Prompt and run these commands:

```cmd
cd "D:\AI-Powered Learning & Productivity Assistant for Beginner Developers\ai-learning-assistant"

mkdir frontend

cd frontend

npm create vite@latest . -- --template react

npm install

npm install react-router-dom

npm run dev
```

### **STEP 4: Verify It's Running**
- Should open at: http://localhost:5173
- You'll see the Vite + React welcome page
