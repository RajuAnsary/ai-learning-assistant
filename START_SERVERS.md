## 🎯 **START FRONTEND (Choose One Method)**

### **Method 1: Using Batch File (Easiest)**
1. Open File Explorer
2. Navigate to: `ai-learning-assistant\frontend`
3. Double-click: `start-frontend.bat`
4. Wait for browser to open automatically

### **Method 2: Using Command Prompt**
1. Press `Win + R`
2. Type: `cmd` and press Enter
3. Copy and paste this command:
```cmd
cd /d "D:\AI-Powered Learning & Productivity Assistant for Beginner Developers\ai-learning-assistant\frontend" && npm start
```
4. Press Enter
5. Wait for browser to open

### **Method 3: Using PowerShell**
1. Press `Win + X` and select "PowerShell"
2. Copy and paste this command:
```powershell
Set-Location "D:\AI-Powered Learning & Productivity Assistant for Beginner Developers\ai-learning-assistant\frontend"
npm start
```
3. Press Enter
4. Wait for browser to open

## ⚠️ **TROUBLESHOOTING**

### **If Frontend Won't Start:**

**Problem**: "react-scripts not found"
**Solution**: Reinstall dependencies
```cmd
cd "D:\AI-Powered Learning & Productivity Assistant for Beginner Developers\ai-learning-assistant\frontend"
npm install
npm start
```

### **If Backend Connection Fails:**

**Check Backend Status:**
1. Open browser
2. Go to: http://localhost:3001/health
3. You should see: `{"status": "OK", ...}`

**If Backend Not Running:**
```cmd
cd "D:\AI-Powered Learning & Productivity Assistant for Beginner Developers\ai-learning-assistant\backend"
npm start
```
